# Mod-load-order-darktide
A lightweight and optimized tool to organize and manage mod load order for Warhammer 40,000: Darktide.

## Features
- Optimized mod loader for faster performance
- Small file size for minimal storage usage
- Easy-to-use drag-and-drop interface

## Installation
1. Download `mod_order_loader.exe`.
2. Drag and drop it into your game’s mod folder.

## Usage
1. Place `mod_order_loader.exe` in your mod folder.
2. Run the executable to automatically generate your mod load order.
3. When you add new mods to the mod folder re-run the mod load order.exe.
